# Vulnerability Disclosure Policy
Policy: https://datacendia.com/trust.html#vulnerability-disclosure

# We appreciate responsible disclosure of security vulnerabilities.
# Please report security issues to security@datacendia.com.
# We commit to acknowledging reports within 48 hours and providing
# regular updates on remediation progress.

# Safe Harbor
# We will not pursue legal action against security researchers who:
# - Make good faith efforts to avoid privacy violations and data destruction
# - Only interact with accounts they own or have explicit permission to test
# - Report vulnerabilities promptly and do not publicly disclose before remediation
